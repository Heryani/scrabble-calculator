# Scrabble App

A React application that serves as a frontend for a Scrabble game. It is a single page app with features such as:

- Input a word of up to 10 letters
- Calculate the score in real-time
- Reset the tiles
- Save the score
- View top 10 scores

## Technologies Used

- React
- Vite
- Typescript
- Axios
- React-icons
